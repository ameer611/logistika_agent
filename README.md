## Installation Guide
1. Clone the repo
2. Enter root folder
3. Get API key from https://console.groq.com/keys
4. Create  `.env` file according to `.env.example` file
5. Put the API key to `GROQ_API_KEY` on `.env`
6. Run the following command:
```bash
docker compose up --build
```
7. Done, Relax:blush:
